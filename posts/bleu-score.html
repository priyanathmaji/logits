BLEU Score

Blue Playground

BLEU (Bilingual Evaluation Understudy) is one of the most commonly used automatic evaluation metrics for Neural Machine Translation (NMT). It measures how closely a candidate translation matches a set of reference translations using n-gram overlap. BLEU balances precision of n-grams with a brevity penalty to discourage overly short translations. In this post, we walk through a step-by-step example to compute BLEU for a single sentence.

Consider a candidate translation:

c = "the love can always do"

and reference translations:

Step 1: Compute modified n-gram precisions

For unigrams (1-grams):

  • Candidate unigrams: the, love, can, always, do
  • Clipped counts: the=0, love=1, can=1, always=1, do=0
  • Total candidate unigrams = 5

$ p_1 = \frac{3}{5} = 0.6$

For bigrams:

  • Candidate bigrams: the love, love can, can always, always do
  • Clipped counts: 0,1,1,0
  • Total candidate bigrams = 4

$ p_2 = \frac{2}{4} = 0.5$

For trigrams:

  • Candidate trigrams: the love can, love can always, can always do
  • Clipped counts: 0,1,0
  • Total candidate trigrams = 3

$ p_3 = \frac{1}{3} = 0.333$

For 4-grams (using smoothing):

  • Candidate 4-grams: the love can always, love can always do
  • Clipped counts (smoothed): 1/3 each (same as previous)

$ p_4 = \frac{1}{3} = 0.333$
