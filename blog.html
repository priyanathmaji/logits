BLEU Score

BLEU (Bilingual Evaluation Understudy) is one of the most commonly used automatic evaluation metrics for Neural Machine Translation (NMT). It measures how closely a candidate translation matches a set of reference translations using n-gram overlap. BLEU balances precision of n-grams with a brevity penalty to discourage overly short translations. In this post, we walk through a step-by-step example to compute BLEU for a single sentence.
